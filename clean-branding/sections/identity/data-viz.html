Guidelines
  • Always start Y-axis at zero (unless there's a valid reason)
  • Use clear, descriptive labels—avoid abbreviations
  • Include data source and date range
  • Ensure sufficient contrast for accessibility
  • Use rounded corners (4px) on bar charts
  • Line weight: 2px for primary data, 1px for secondary
