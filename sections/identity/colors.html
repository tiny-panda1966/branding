A colour palette in branding is a curated set of colours representing your brand's identity and helps create a recognisable, cohesive look across all materials. It typically includes primary colours, which form the core of the brand's look and are used most frequently; secondary colours, which complement the primary colours and add flexibility; and accent colours, used sparingly to emphasise key elements like buttons or calls to action.
